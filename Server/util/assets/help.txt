
FTP Help
This document provides a list of available commands for the server application, along with their usage and descriptions.
Available Commands:

login Usage: login <username@password>
--------------------------------------------------
upload Usage: upload <file_path>
--------------------------------------------------
download Usage: download <file_name>
--------------------------------------------------
mkdir Usage: mkdir <directory_name>
--------------------------------------------------
rmdir Usage: rmdir <directory_name>
--------------------------------------------------
cd Usage: cd <directory_name>
--------------------------------------------------
resume Usage: resume <file_name>
--------------------------------------------------
rename Usage: rename <old_name> <new_name>
--------------------------------------------------
list Usage: list
--------------------------------------------------
help

Notes:
- Ensure that you are logged in before attempting to use commands that require authentication (e.g., upload, download, etc.).
- Commands are case-insensitive, but it is recommended to use lowercase for consistency.
- If you encounter any issues, please refer to the command usage for proper syntax.

For further assistance, please contact the server administrator.
